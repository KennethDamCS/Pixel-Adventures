# Pixel Adventures <br/>
Super Mario Bros-like 2D platformer developed using Unity and C# <br/>
Player Controls: <br/>
-Left: A <br/>
-Right: D <br/>
-Jump: SPACE <br/>
Assets used: https://assetstore.unity.com/packages/2d/characters/pixel-adventure-1-155360
